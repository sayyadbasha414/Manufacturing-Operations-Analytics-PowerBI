🏭 Manufacturing Operations Analytics Dashboard
📌 Project Overview

This Power BI project provides a comprehensive analysis of manufacturing operations through interactive dashboards and KPIs. The solution helps stakeholders monitor production efficiency, product quality, machine performance, downtime, and employee productivity to support data-driven decision-making.

🎯 Business Objective

The objective of this project is to:

Monitor production performance across multiple plants.
Track product quality and defect trends.
Analyze machine utilization and downtime.
Evaluate employee productivity and efficiency.
Provide actionable insights for operational improvement.
